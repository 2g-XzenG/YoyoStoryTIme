## Setup and Run Instructions

1. **Activate Virtual Environment**:
   - On macOS/Linux:
     ```bash
     source venv/bin/activate
     ```
   - On Windows:
     ```bash
     .\venv\Scripts\activate
     ```

2. **Run the Application**:
   ```bash
   python app.py

3. **Access the App: Open your browser and go to http://127.0.0.1:5000**
